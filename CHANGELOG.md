# Changelog

## [0.3.0] — 2026-07-02

### Added

- Звук без ассетов: мини-синтезатор на Web Audio (`src/audio/sfx.ts`) — шлепок тапка, промах, пах-комбо, Q/E/R, волны, победа/поражение; `M` — мьют
- Game feel: хит-стоп при отбитии, тряска камеры на промахах/поражении и при высоком хаосе, squash-анимации задач
- Пиксельный Паша (`src/entities/pixelArt.ts`): борода, футболка с принтом, малыш и дочка прямо в руках; лицо реагирует на энергию (бодрый → устал → d[e]ad)
- Пауза `P`/`Esc` с корректной заморозкой кулдаунов
- Спавн-директор: задачи чаще прилетают по просевшей шкале — проигрыш из-за чистого RNG исключён
- Архетипы задач: быстрые (плач), жирные в 2 удара (дедлайн), зигзаг (курьер, Wi-Fi)
- Именованные мини-волны: «Созвон-пятиминутка» (45с), «Час курьеров» (95с), «Детский бунт» (140с)
- Хэдбатт: прицел над головой — удар башкой с бонк-звуком, работает с занятыми руками
- Интро: Паша вбегает на позицию под топоток, дела летят только после прибытия
- Курсор-членик (SVG data-URI, целишься кончиком)
- Паша — дизайнер: пул задач вырос до 23 («правки v8», «сделай красиво», «лого побольше», «скинь референсы», «фигма лагает»…)
- Малыш и дочка читаются на руках: свёрток с личиком и девочка с косичками

### Changed

- Визуальный стиль: кислотный пиксель-брутализм — чёрный фон без сетки, один доминантный акцент, плоские панели и кнопки, задачи-стикеры с толстой цветной рамкой
- Выбор конечности детерминирован: прицел ниже пояса — ноги, выше — руки; вместо постоянной арки хитбокса — тонкий ретикл
- Предметы стали значимыми: Тапок = +20% очков, Погремушка = +30% радиуса удара
- Кривая сложности плавная (до 150с) вместо трёх ступеней
- Пропуски сильнее топят Пашу в куче (5px вместо 3px)

## [0.2.2] — 2026-07-01

### Added

- Общий neon/noir UI-kit в `src/ui/theme.ts`: палитра, фон, панели, теги и кнопки
- Визуально цельные экраны Menu, Game, Result и Leaderboard
- HUD с численными значениями шкал, режимом прицела и готовностью Q/E/R
- Enter-start в меню и Enter-retry на экране результата
- Popup-фидбек для действий Q/E/R, если действие ещё на кулдауне

### Changed

- Задачи теперь выглядят как летящие неоновые карточки с цветовой маркировкой типа
- Паша получил контур, лицо, ауру и более читаемые состояния занятых/свободных рук
- Хитбоксы, defeat sequence и куча дел приведены к новой палитре
- README и `docs/game-design.md` дополнены визуальным стилем и UX-заметками
