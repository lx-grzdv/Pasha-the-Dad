# Геймдизайн

## Концепция

Паша в центре экрана. Руки заняты малышом и дочкой — по умолчанию бьёт **ногами**. Освобождение рук (Q/E/R) — рискованный трейдофф.

## Шкалы (0–100, кроме хаоса 0–10)

| Шкала | Game over при 0 | Пассивный drain |
|-------|-----------------|-----------------|
| Малыш | да | да, быстрее при подбросе |
| Дочь | да | да, при саде |
| Работа | да | да, быстрее в туалете |
| Энергия | да | да |
| Хаос | нет | растёт со временем |

## Действия

| Клавиша | Эффект | Длительность | Кулдаун |
|---------|--------|--------------|---------|
| Q | Левая рука свободна | 6с | 8с |
| E | Правая рука свободна | 18с | 20с |
| R | Обе руки (туалет) | 6с | 12с |

## Конечности

| Состояние | Limb | Радиус / сила |
|-----------|------|---------------|
| Обе заняты | feet | слабый |
| Левая свободна | leftHand | средний |
| Правая свободна | rightHand | средний |
| Туалет | bothHands | сильный |

## Куча дел

Пропущенные задачи прилипают у ног. Каждые 2 пропуска Паша опускается на 3px. При поражении — лавина + «D[e]ad».

## Предметы

- **Тапок** — универсальный
- **Погремушка** — +25% baby, -15% work

## Длительность

180 секунд. Победа — дожить до нуля таймера.

## Scoring

```
score = Σ(taskPoints × combo) + survival×10 + balanceBonus + (win ? 500 : 0)
```

Комбо: x1.2 @5, x1.5 @10, x2 @20
